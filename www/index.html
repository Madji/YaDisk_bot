@YaDisk_bot

@yadisk_bot - телеграм бот, который загружает все, что вы ему посылаете, на Яндекс.Диск.

  • @yadisk_bot не потребует ваш пароль на Яндекс.Диск, ему будет достаточно токена, который выдаст Яндекс, для работы с этим приложением.
  • @yadisk_bot не сможет увидеть все ваши файлы в Яндекс.Диске. Ему будет доступна только отдельная папка, созданная специально для него.
  • Вы сможете найти эту папку по следующему пути: "Приложения->Файлы из Телеграма."
